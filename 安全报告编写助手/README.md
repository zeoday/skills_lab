# 安全报告编写助手

## 项目简介

安全报告编写助手是一款专注于网络安全领域的智能报告生成工具，能够基于漏洞编号、名称或安全扫描工具报告，自动生成符合行业标准的漏洞分析报告，包含漏洞背景、漏洞描述、漏洞原理、漏洞利用条件和修复要求等核心内容，帮助安全从业者快速生成标准化、专业的安全报告。

## 功能特性

- **漏洞信息解析**：支持输入漏洞编号、名称或导入安全扫描工具报告
- **漏洞分析与评估**：对漏洞进行深入分析，提供风险评估和优先级建议
- **多模板选择**：提供通用模板和行业特定模板（金融、医疗、政府、制造业等）
- **多漏洞总分结构**：支持多个漏洞的综合分析报告
- **CVSS评分自动计算**：基于CVSS 3.1标准自动计算漏洞评分
- **修复建议生成**：根据漏洞类型和严重程度提供针对性修复建议
- **数据导入与集成**：支持对接漏洞扫描工具的导出数据

## 安装与使用

### 安装

1. 克隆或下载本项目到本地
2. 在 Trae CN 中导入安全报告编写助手 Skill
3. 配置触发词（如「生成 SQL 注入漏洞报告」「生成 XX 漏洞 CVSS3.1 评分报告」）

### 使用流程

#### 单个漏洞报告生成
1. 输入漏洞信息（漏洞编号、名称或扫描工具报告）
2. 选择报告模板（从 `templates` 目录中选择合适的模板文件）
3. 一键生成标准化漏洞报告
4. 根据需要修改和定制报告内容
5. 导出或分享报告

#### 多个漏洞的总分结构报告生成
1. 输入多个漏洞信息（可列出漏洞编号、名称或上传扫描工具报告）
2. 系统自动分析所有漏洞，生成漏洞概述和风险评估
3. 选择多漏洞总分结构模板（`templates/multi_vulnerability_template.md`）
4. 一键生成总分结构的漏洞报告
5. 查看报告概述和每个漏洞的详细分析
6. 根据需要修改和定制报告内容
7. 导出或分享报告

## 模板说明

### 通用模板
适用于大多数场景的标准漏洞报告，包含漏洞背景、描述、原理、利用条件和修复要求等核心部分。

### 行业特定模板
- **金融行业模板**：符合金融行业合规要求的报告格式，包含监管合规影响、业务影响评估等
- **医疗行业模板**：满足医疗数据安全法规的报告格式，包含患者数据保护、医疗行业特殊考虑等
- **政府行业模板**：符合政府部门安全标准的报告格式，包含政务信息保护、等级保护要求等
- **制造业模板**：针对工业控制系统的报告格式，包含生产连续性考虑、工业控制系统安全标准等
- **教育行业模板**：符合教育行业合规要求的报告格式，包含学生数据保护、教育系统安全考虑等
- **能源行业模板**：符合能源行业合规要求的报告格式，包含能源系统安全、关键基础设施保护等
- **交通行业模板**：符合交通行业合规要求的报告格式，包含交通系统安全、公共安全考虑等

### 场景特定模板
- **渗透测试模板**：针对渗透测试场景的报告格式，包含测试方法、攻击路径、漏洞验证等
- **YouTrack提报模板**：用于在YouTrack中提报安全需求的模板，包含需求背景、需求来源、需求描述、验收标准等
- **简化版YouTrack模板**：简化的YouTrack提报模板，适合快速提报安全需求
- **表格版YouTrack模板**：使用表格形式的YouTrack提报模板，便于清晰展示漏洞信息

### 多漏洞总分结构模板
用于多个漏洞的综合分析报告，包含漏洞概述、风险评估、修复时间线等。

## 资源说明

### 漏洞信息检索资源
包含官方漏洞数据库、厂商安全公告、安全漏洞平台和漏洞扫描工具等资源站点，帮助用户快速查询漏洞信息。
